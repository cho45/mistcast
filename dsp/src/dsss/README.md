# DSSS (Slow) Mode

DSSS (Direct Sequence Spread Spectrum) モードは、計算リソースを極限まで削った軽量・柔軟な実装です。

## 技術仕様

| 特徴 | DSSS (Slow) |
| :--- | :--- |
| **変調方式** | DSSS + DQPSK |
| **伝送効率** | 2 bits / symbol |
| **インターリーバ** | 12×29=348 bits |
| **拡散率 (SF)** | 15 (M系列) |
| **チップレート** | 8,000 cps |
| **キャリア周波数** | 15,000 Hz |
| **実効速度 (Goodput)** | **~ 352 bps** (1 pkt/frame) |
| **マルチパス対策** | N/A |

### 物理層へのマッピング

```
パケット (21B) → FECエンコード (174→348 bits) → インターリーバ (348 bits, paddingなし)
                                     ↓
                    DQPSK変調 (2 bits/symbol) → 174 symbols
                    ↓
                    DSSS拡散 (SF=15) → 2,610 chips
```

### 同期捕捉 (Synchronization)

#### プリアンブル（Preamble）

| モード | 系列 | 拡散率 | パターン | 特徴 |
| :--- | :--- | :--- | :--- | :--- |
| **DSSS (Slow)** | M系列 | SF=15 | [M, -M] | ノンコヒーレント相関による軽量な自己相関 |

- **最終シンボル反転**: 最後のシンボルを反転させ、同期の位相曖昧さを排除
- **繰り返し回数**: `preamble_repeat` パラメータで調整可能（デフォルト2回）

#### Sync Word

| モード | Sync Word長 | 変調方式 | 拡散系列 |
| :--- | :--- | :--- | :--- |
| **DSSS (Slow)** | 16ビット | DBPSK | M系列 (SF=15) |

## トレードオフと評価

- **ノイズ・自己干渉耐性**: 低い。全データが位相(DQPSK)に依存するため、帯域制限による自己干渉(ISI)の位相ジッタに弱く、ベースラインエラーが生じます。
- **マルチパス耐性**: 低い。時間領域のシンプルな追従のみ行うため遅延波による波形歪みに弱いです。
- **計算負荷 (CPU)**: **低い**。1つの相関器のみで動作するため軽量です。
- **パラメータの柔軟性**: **高い**。M系列の長さ(SF)を自由に変更しやすく、感度と速度のトレードオフを容易に設計できます。
